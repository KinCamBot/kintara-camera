KinCam Cinematic Camera

First-person, free-cam photos, over-the-shoulder, and one-click HYPE reels, right inside Kintara.

How to install

1
Get Tampermonkey, a free browser extension (one-time setup). Install it from the official Chrome Web Store: Get Tampermonkey →
2
Turn on user scripts. Right-click the Tampermonkey icon in your toolbar and click Manage extension: Then scroll down and switch Allow User Scripts on: Chrome, Edge and Brave only. Firefox users can skip this step.
3
Install the camera mod. Click below, then hit Install on the Tampermonkey tab that opens: ⬇  Install Kintara Camera
4
Open kintara.gg/play and log in. The camera panel appears at the top-right. 🎥

What it does

  • First-person, Play, and Over-the-shoulder play cameras
  • Free Cam for photographing your character from any angle
  • Cinematic Auto-Pan moves + a 20s auto HYPE reel that records itself
  • Clean photo & clip capture, hide-HUD toggles, crosshair toggle
